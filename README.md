# Identifo for iOS

An API of universal authentication framework for iOS.

Check [main repo](https://github.com/MadAppGang/identifo) for general project overview. 

## Requirements

- iOS 11.0+
- Xcode 10.3+
- Swift 5+
- Carthage 0.33.0+

## Setup

### CocoaPods

```ruby
pod 'identifo-ios', '~> 0.1.0'
```

### Carthage

```ogdl
github "MadAppGang/identifo-ios" "0.1.0"
```
